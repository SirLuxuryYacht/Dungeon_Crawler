## Description
This is a one-person's attempt at creating a King's Filed inspired dungeon crawler with level progression, a working quest system and custom textures, models and music.

## How to use
As is, Godot v4.6.2 is required (or at least recommended) to run these files. At engine startup, place the "project.godot" into the engine window and run it (like any other project). Make sure the files and the "project.godot" are in the same folder.
